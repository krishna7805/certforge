
# # Certificate Generator

A professional, browser-only(Web-Based) Bulk certificate generator for creating high-quality certificates with zero coding. Upload a template, add names, style text, and export in bulk.

## 🚀 Try It Now
**[👉 Live Demo - Try Here!](https://krishna7805.github.io/certforge/)**

## ✨ Key Features

- **🎨 Visual Design** - Drag & drop text positioning, real-time preview
- **📄 Template Support** - Upload PNG/JPG certificate backgrounds  
- **👥 Bulk Generation** - Enter multiple names, generate all certificates at once
- **💾 Multiple Exports** - PNG, PDF, and ZIP downloads
- **🎯 No Backend** - Runs entirely in your browser
- **📱 Print Ready** - High-resolution, professional quality output

## 📖 Quick Usage

### 1. **Upload Template**
- Click "Choose File" and select your certificate template (PNG/JPG)

### 2. **Add Names** 
- Enter participant names in the textarea (one per line)

### 3. **Style & Position**
- Drag text to desired position
- Adjust font, size, color, and alignment using right panel controls

### 4. **Export**
- **Single:** Export PNG/PDF of current certificate
- **Bulk:** Export All (ZIP) for all participants

## 🎛️ Interface

| Left Panel | Center | Right Panel |
|------------|---------|-------------|
| Template Upload<br>Name Input<br>Export Buttons | Live Preview Canvas<br>Draggable Text<br>Name Navigation | Font Controls<br>Styling Options<br>Position Tools |

## 💡 Best Practices

- Use high-resolution templates (1200x800px minimum)
- Test text positioning with longest participant names
- Use readable fonts and sufficient color contrast
- Export as ZIP for large batches (10+ certificates)
