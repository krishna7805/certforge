### Step 1: Upload Template

3. Make final adjustments as needed

- **Export PDF** - Print-ready PDF of current certificate
- **Export All (ZIP)** - All certificates as PNG files in a ZIP

## ğŸ’¡ Tips & Best Practices

### Template Preparation
- Use high-resolution templates (minimum 1200x800px)
- PNG format recommended for templates with transparency
- JPG format works well for solid background templates

### Text Styling
- Use readable fonts for professional appearance
- Ensure sufficient contrast between text and background
- Test with longest participant names to avoid overflow

### Bulk Generation
- Keep participant names consistent in formatting
- Remove extra spaces or special characters
- Preview multiple names before bulk export

### File Management
- Individual exports: For single certificates or small batches
- ZIP export: For large batches (10+ certificates)
- PDF format: Best for printing and official documents
